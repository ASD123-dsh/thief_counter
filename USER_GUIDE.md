## 1. 基础操作

*   **模式切换**：点击左上角的“模式”按钮，可在“普通”、“科学”、“程序员”三种模式间切换。
*   **窗口置顶**：点击右上角的“图钉”图标，可将计算器固定在屏幕最顶层。
*   **日/夜主题**：点击右上角的“月亮/太阳”图标，切换深色或浅色主题，适应不同光线环境。
*   **调整大小**：拖动窗口右下角的手柄即可调整窗口大小。
*   **移动窗口**：按住标题栏区域（顶部空白处）即可拖动窗口。

## 2. 计算模式详解

### 2.1 普通模式
满足日常计算需求。
*   **支持运算**：加减乘除、百分比 (`%`)、平方根 (`√`)、平方 (`x²`)、倒数 (`1/x`)。
*   **历史记录**：右侧列表实时显示计算历史，方便回溯。

### 2.2 科学模式
适用于工程与数学计算。
*   **高级函数**：三角函数 (`sin`, `cos`, `tan`)、双曲函数 (`sinh`...)、对数 (`log`, `ln`)、阶乘 (`n!`)、指数 (`Exp`) 等。
*   **角度切换**：顶部下拉框支持在“角度 (deg)”与“弧度 (rad)”之间切换。
*   **常数**：支持插入 `π` 和 `e`。

### 2.3 程序员模式
专为开发者设计。
*   **进制转换**：支持 **HEX** (十六进制)、**DEC** (十进制)、**OCT** (八进制)、**BIN** (二进制) 实时切换与显示。
*   **位运算**：支持与 (`AND`)、或 (`OR`)、异或 (`XOR`)、非 (`NOT`)、左移 (`<<`)、右移 (`>>`)。
*   **双操作数**：提供 A、B 两个输入框，方便进行二元位运算。
*   **记忆功能**：提供明确的 `MC`、`MR`、`M+`、`M-` 按钮进行数值存储。
*   **位宽设置**：支持自定义位宽（默认为 32 位）。

## 3. 摸鱼阅读器（核心特色）

这是一个深度集成在计算器内部的 TXT 阅读器，具备极高的隐蔽性。

### 3.1 如何开启
阅读器功能默认隐藏，首次使用需要解锁：
1.  切换到 **普通模式**。
2.  在计算器输入框输入密钥 **`666888`**。
3.  点击 **`=`** 按钮。
4.  此时顶部会出现一个 **“设置”** 按钮。
5.  点击“设置”，输入你的 TXT 小说文件夹路径，并设置透明度。

*提示：配置完成后，以后可以直接使用快捷键 `Ctrl + M` 快速进入阅读模式。*

### 3.2 阅读操作
进入摸鱼模式后，计算器左侧区域将变身为阅读器：
*   **翻页**：
    *   鼠标滚轮
    *   方向键 `↑` / `↓` / `←` / `→`
    *   `PageUp` / `PageDown`
    *   **`W` / `S`** (WASD 键位，方便左手操作)
*   **跳转**：双击页码标签可输入页码直接跳转。

### 3.3 极简悬浮窗
如果你觉得计算器窗口还是太显眼，可以开启“极简模式”：
*   **开启方式**：在摸鱼模式下，连续点击 **4 次 `=` 按钮**。
*   **效果**：会弹出一个独立的、极小的、无边框的透明阅读条。
*   **特性**：
    *   **自动悬隐**：鼠标移开后会自动隐藏（几乎完全透明），鼠标靠近时才会浮现。
    *   **完全透明**：在设置中将透明度设为 `1`，可获得仅显示文字的完全透明背景效果。

### 3.4 一键伪装
遇到紧急情况？
*   按下 **`Esc`** 键。
*   **效果**：立即退出阅读模式，并在界面上随机显示一组复杂的**数学公式**（如麦克斯韦方程组、微积分公式等），完美伪装成正在进行高深运算的样子。

## 4. 内置小游戏（隐藏彩蛋）

工作累了？计算器里还藏着几款经典游戏。

### 4.1 如何开启
游戏入口默认隐藏，需要解锁：
1.  切换到 **科学模式**。
2.  在输入框输入密钥 **`666888`**。
3.  点击 **`=`** 按钮。
4.  此时顶部会出现一个 **“选择”** 按钮。
5.  点击“选择”，即可启动以下游戏：
    *   **文字 2048**
    *   **文字贪吃蛇**
    *   **文字扫雷**
    *   **字符五子棋**

### 4.2 通用特性（悬隐）
所有游戏窗口均采用“纯文本/字符”风格，且具备强大的**悬隐功能**：
*   **移出隐藏**：鼠标移出游戏窗口范围，窗口会立即变透明（几乎不可见）。
*   **移入显示**：鼠标移入窗口区域并停留片刻（约 1.5 秒），窗口会重新浮现。
*   **拖动**：按住窗口任意区域（或特定拖动区）即可移动。

### 4.3 游戏操作指南
*   **2048**：使用方向键或 `WASD` 移动方块合并。`R` 键重开。
*   **贪吃蛇**：使用方向键控制移动。`R` 键重开。
*   **扫雷**：左键揭开，右键标记旗帜。
*   **五子棋**：方向键/WASD 移动光标，**空格键**落子。

## 5. 快捷键汇总

| 按键 | 作用 | 适用范围 |
| :--- | :--- | :--- |
| **Ctrl + M** | 快速进入/恢复摸鱼阅读模式 | 全局（需先配置路径） |
| **Esc** | 退出摸鱼模式（开启伪装） / 关闭游戏窗口 | 摸鱼模式 / 游戏窗口 |
| **W / S** | 上一页 / 下一页 | 摸鱼模式 |
| **连续 4 次 `=`** | 唤出极简悬浮阅读窗 | 摸鱼模式 |
| **R** | 重置游戏 | 游戏窗口 |
| **空格** | 落子 | 五子棋 |
